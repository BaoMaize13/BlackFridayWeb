# BlackFridayWeb Frontend

Production-oriented frontend for oversell monitoring, distributed locking, concurrency simulations, observability, and technical reporting.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Environment

Copy `.env.example` to `.env` and adjust the backend URL if needed.

Core variables:

- `VITE_API_BASE_URL`
- `VITE_AUTH_LOGIN_PATH`
- `VITE_AUTH_VALIDATE_PATH`
- `VITE_SESSION_STORAGE_KEY`
